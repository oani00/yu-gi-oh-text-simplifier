# Ygo text simplifier

I absolutelly HATE the wording of yugioh, it is like a single line, font size 1, were every character counts. SO to ease understanding, I vibe coded a simple html that just puts collors in common ygo separators like ; or , that, for me, makes reading ~~possible~~ better.
